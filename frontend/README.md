# Buenos Tickets - Frontend

Buenos Tickets 티켓 판매 시스템의 프론트엔드 웹 애플리케이션입니다.

## 🚀 개발 서버 실행 방법 (추천)

### npx로 설치 없이 실행 ✨

가장 간단하고 빠른 방법입니다! 별도의 전역 패키지 설치 없이 바로 서버를 실행할 수 있습니다.

```bash
# frontend 폴더로 이동
cd frontend

# npx로 serve 실행 (포트 8000)
npx serve -p 8000
```

### 서버 실행 후

브라우저에서 다음 주소로 접속:

- **판매자 페이지**: http://localhost:8000/merchant.html
- **고객 페이지**: http://localhost:8000/customer.html
- **테스트 페이지**: http://localhost:8000/customer-test.html
- **x402 페이지**: http://localhost:8000/x402.html
